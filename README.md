# Todos In Vue.js & Vuex
A todolist project for practicing new library.

## Install Dependencies

```
npm i
```

## Run Mock API Server

```
// http://localhost:3000/tasks
json-server -w db.json
```

## Start Devlopment Server

```
// http://localhost:8080
npm start
```

## Build Bundle JS & Run Web Server

```
npm run build
```
